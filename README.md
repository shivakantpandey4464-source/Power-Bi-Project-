## Project Overview
Is project mein maine India ki GDP ka data analyze kiya hai aur ek interactive Power BI dashboard banaya hai. Isse humein India ki economic growth aur trends samajhne mein madad milti hai.

## Key Insights
* **Yearly Trends:** Pichle kuch saalon mein GDP mein kaise badlav aaya.
* **Sector Analysis:** Agriculture, Industry, aur Services ka yogdaan.
* **Growth Rate:** GDP growth percentage ka analysis.

## Tools Used
* **Power BI Desktop:** Data visualization aur reporting ke liye.
* **Excel/CSV:** Raw data source ke liye.

## How to View the Report
1. Is repository se `Indian_GDP_Reports.pbix` file download karein.
2. Ise **Power BI Desktop** mein open karein.
